# Syntecxhub_Project_2_Statistical_Plots
<h1>Task No .2</h1>
<h2>Project_2-->Statistical Plots and Distribution Analysis</h2>

<i>Overview
<br>
This project focuses on analyzing a sales dataset using statistical visualization techniques. The goal is to understand data distribution, identify outliers, and compare patterns across different variables using Python.<i>
<br><br><br>
Dataset<br>
The dataset contains information related to:<br>
- Sales<br>
- Profit<br>
- Category<br>
- Geography<br>
- Order details<br>
<br><br><br>
 Tools & Libraries Used<br>
- Python<br>
- Pandas<br>
- Matplotlib<br>
- Seaborn<br>
<br><br><br>
 Data Preprocessing<br>
- Checked null vales,missing data<br>
- Checked date formats,duplicate values<br>
<br><br>
 Visualizations Performed<br><br>
1. Histogram (Sales Distribution)<br>
- Used to understand how sales values are distributed:<br>
- Most values are in lower range<br>
- Right-skewed distribution observed<br>
<br>
2. KDE Plot (Profit Distribution)<br>
- Shows smooth distribution of profit values.<br>
- Majority of values near zero<br>
- Presence of both profit and loss<br>
<br>
3. Boxplot (Sales)<br>
- Used to detect spread and outliers.<br>
- Large number of outliers present<br>
- Data is highly skewed<br>
<br>
4. Boxplot (Category vs Sales)<br>
- Comparison of sales across categories.<br>
- Some categories show higher variability<br>
- Others have consistently low sales<br>
<br>
5. Scatter Plot (Sales vs Profit)<br>
- Shows relationship between sales and profit<br>
- Positive relationship observed<br>
- Some transactions result in loss<br>
- Few extreme high-value points<br>
<br><br>
 Key Insights<br>
- Sales data is highly right-skewed<br>
- Most transactions have low sales and profit<br>
- Outliers significantly affect the distribution<br>
- Higher sales generally lead to higher profit<br>
- Profit varies across categories<br>
<br><br>
  Challenges Faced<br>
- Crowded plots due to many categories<br>
- Extreme outliers affecting visualization clarity<br>
<br><br>
 Improvements Applied<br>
- Used log scale for better visibility<br>
- Switched to category-based comparisons instead of geography<br>
<br><br>
 Conclusion<br>
The project helped in understanding how data distribution, outliers, and relationships between variables can be analyzed using visualization techniques. These insights can be useful for making better business decisions.
